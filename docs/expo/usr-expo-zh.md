# USR EXPO（中文）

## 內容摘要

突破疫情封鎖 溫暖白色巨塔

在2020年疫情封鎖下，人人保持社交距離，恐懼似乎拉遠了人之間的距離，不過文藻WHC-USR 團隊以愛心搭起連結的橋樑， 讓高雄的關懷陽光突破疫情的恐懼圍籬，將旗山國小孩童與文藻青年的熱情，傳遞到遙遠的柬埔寨，溫暖了柬埔寨學童的心靈。

旗山國小位居高雄偏遠山區，為協助防疫與環保觀念向下紮根，文藻USR大學社會責任團隊七位老師，帶領修習「國際專業服務學習領導」課程的27名學生，從收集資料、教學設計、教材製作、教學演練等，師生使出渾身解數，就是希望能夠將衛教英文生活化，以活潑有趣的方式教給小朋友。於五月前往旗山國小，透過英文教導小朋友認識登革熱、學習垃圾分類、製作簡易口罩、包裝口罩套等，活動中也由旗山國小學生協助包裝口罩套，並寫防疫友誼卡片，寄給柬國的偏鄉小學生。

因疫情影響的關係，文藻WHC-USR團隊在柬埔寨的夥伴-綠雨傘(Green Umbrella,GU)小學的英文課程幾乎停擺，當地小朋友們只能待在家裡，苦無學習機會。文藻國際志工從挑選英文教材開始一系列流程，勇敢的開創線上英語教學的服務，運用網路會議軟體為教學工具，設計活潑有趣的線上英文課程，每週一次讓柬國小學生用手機學習英文。合作的KKS小學則有12位小學生參加此計畫，年齡層從9歲到11歲，他們只學過基本的英文字母，由學校協助買一支25元美金的手機，然後用手機在家裡學習，這也是柬國學生們英文線上課程的初體驗。文藻國際英語志工們紛紛使出渾身解術，用各種手勢、聲音、表情及簡單的線上互動活動，讓英文零起點的柬國小學生，可以很快進入狀況，從一開始的害羞，到後來能以完整的英文句子自信回應。孩子們每次學習英文時的笑容，不僅拉近了我們的距離，更牽起了我們的情誼。雖然短時間內學會的英文是有限的，但引發他們對英文學習的興趣會帶來無限可能。而我們一直深信，教育是個以生命感動生命的歷程，不管是被教的孩子們，還是教人的我們，都在彼此的心靈上互相感動，而這份感動正是教育的溫度。

五月份由文藻青年志工及旗山國小學生共同完成的環保口罩套及祝福卡片，經過遙遠曲折的海上旅程，已經安全送抵柬埔寨，當地的小朋友們及老師們收到時都非常高興，創辦人Sokrath表示，接收到的不只是新的防疫物資，更感受到來自台灣文藻同學及旗山國小同學們滿滿的愛心，而這份愛，也會在柬埔寨持續蔓延。如圖四及圖五所示。

除了關懷偏鄉及柬埔寨孩童，在疫情緊張的情況下，針對移工跟新住民防疫衛教需求，WHC-USR團隊提供一套包含五種語言LINE以及IG即時資訊系統，由文藻擅長英、越、印、泰語的老師加入值班服務，每日將最新防疫資訊轉譯為該國的語言，同時連結小港醫院的醫療衛教團隊，提供各工業區外籍移工專業衛教資訊的諮詢，發揮語言專業共同為台灣防疫把關。

「溫暖白色巨塔的小螺絲釘－文藻國際志工共創就醫無障礙」USR社會實踐計畫(WHC-USR)已執行三年，前兩年主要協助學生前往高雄三家醫院擔任志工，協助新住民及移工克服就醫過程的語言及文化障礙，如圖九所示。也與夥伴醫院前往柬埔寨協助衛教及環保教育服務，如圖十所示。今年因應疫情影響，八月份之前主要調整為新住民、移工及偏鄉小學防疫衛教的服務，並協助柬埔寨的線上英語服務。隨著台灣疫情緩和，將在十月份展開在高雄榮總、義守醫院及小港醫院的志工服務。本計畫發揮本校外語暨國際化的優勢，以關懷語言弱勢族群為出發點，串聯學校及跨界資源，與地方三所醫院合作，推動國際志工專業服務，希冀共創「就醫無障礙、學習無國界」的願景。
